# AI intensive projects

Two files containing AI-intensive software repositories (produced as part of the "Agentic Much?" study). 

Each file contains projects for which the heuristics identified at least 75% of AI-assisted commits, and more than 50 such commits.

- the "older" files is a least of ~250 projects created before 30/08/2025
- the "newer" files is a least of ~500 projects created after 30/08/2025
